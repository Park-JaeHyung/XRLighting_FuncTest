# XR Lighting Designer — 직업체험 콘텐츠

XR로 개발하기 전, PC 환경에서 핵심 기능을 먼저 검증하는 기능테스트 프로젝트입니다.

## 콘텐츠 개요

조명 감독 직업체험 콘텐츠. 사용자는 가상 공간에 조명을 배치하고, 실제 조명 감독이 쓰는 콘솔과
비슷한 패널로 조명을 제어하며, 조명 상태(색상/움직임/앵글 등)를 큐(Cue)로 저장해 큐시트(Cue Sheet)를
편집하고 재생하며 감상할 수 있습니다.

## 핵심 사용자 흐름

```
앱 실행 → 가상 공간 선택 → 조명 배치(사전 정의된 조명 데이터 사용)
   → 조명 조정 후 큐 저장 → 큐시트 저장 → 재생하며 조명 움직임 감상
```

## 도메인 모델

| 개념 | 설명 |
|---|---|
| FixtureProfile (SO) | 조명 종류 정의 (무빙헤드/PAR/스팟 등), 지원 파라미터 및 프리팹 참조 |
| FixtureInstance | 씬에 배치된 실제 조명, 현재 파라미터 상태 보유 |
| LightParameters | 한 시점의 조명 상태값 (Color, Intensity, Pan, Tilt, Beam Angle 등) |
| Cue | 특정 순간의 전체 조명 스냅샷 (Cue Number, Label, Fade Time 포함) |
| CueSheet | Cue의 정렬된 시퀀스 (재생 순서/타이밍) |

## 아키텍처 레이어

```
Control Panel UI (조명 콘솔 모사)
        ↓ 선택된 Fixture에 바인딩
Fixture Layer (FixtureInstance ↔ Light 컴포넌트)
        ↓ Record → 현재 상태 스냅샷
Cue System (CueManager: 생성/저장/보간/재생)
        ↓
Cue Sheet Editor UI (리스트/타임라인 시각화, 재생)
```

- 레이어 간 통신은 이벤트 기반으로 분리 (UI가 Fixture를 직접 참조하지 않음)
- 조명 제어는 `ILightControllable` 인터페이스로 추상화
- 큐 재생 시 보간(Interpolation)은 파라미터 타입별로 분리된 모듈에서 처리
  (Color는 HSV 보간, Pan/Tilt는 최단 각도 보간, Intensity는 선형 보간)
- 배치/선택 입력은 `IPlacementInput`으로 추상화해 추후 XR 컨트롤러 입력으로 교체 가능하게 설계

## MVP 범위

| 단계 | MVP에서 할 일 | 이후로 미루는 것 |
|---|---|---|
| 앱 실행 | 단일 진입 씬 | 로그인, 인트로 연출 |
| 공간 선택 | 프리셋 2~3개 중 선택 | 공간 커스터마이징 |
| 조명 배치 | 사전 정의 FixtureProfile 목록에서 배치 (위치만) | 조명 커스텀 제작, 드래그앤드롭 |
| 큐 저장 | 조정 후 스냅샷 저장 | Undo, 큐 이름/메모 편집 |
| 큐시트 저장 | 큐를 순서대로 리스트에 저장 | 타임라인 시각화, 드래그 재정렬 |
| 재생/감상 | 큐 순서대로 자동 재생, 크로스페이드 | 일시정지, 스크럽, 속도 조절 |

MVP 단계에서는 프리미티브 도형과 Unity 기본 UI로 전체 루프를 완성하는 것을 우선하고,
실제 조명 모델/UI 스킨은 기능 검증 이후에 적용합니다.

## 개발 로드맵

1. **기획/설계** — 콘텐츠 정의, 도메인 모델, 아키텍처, MVP 범위 확정 (현재 단계)
2. **PC 기능테스트 MVP** — 배치→제어→큐→큐시트→재생 전체 루프 동작 확인
3. **PC 고도화** — 실제 에셋 적용, Undo/Redo, 큐시트 타임라인, 저장/불러오기
4. **XR 전환 설계** — 입력/UI를 XR 컨트롤러·월드스페이스 캔버스로 교체 가능하도록 검증
5. **XR 프로토타입** — 헤드셋에서 전체 루프 동작 확인
6. **XR 폴리싱/배포** — 튜토리얼, 연출, 사운드 등 완성도 작업

## 참고 사항

- 실제 물리 조명(DMX/Art-Net 연동)은 현재 범위에 포함되지 않음. 실물 fixture와 연동이 필요해지는
  시점에 별도 검토 (지금은 가상 조명이므로 Unity Light 컴포넌트를 직접 제어)
- 렌더 파이프라인: URP
- 입력: Unity Input System

## 기술 스택

- Unity 6000.0.66f2
- URP (Universal Render Pipeline)
- Unity Input System
- UI Toolkit (제어 패널 / 큐시트 UI 권장)
